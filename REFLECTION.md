## Question 4: How can you apply multithreading concepts in real-world applications?

**Your Answer:**

Multithreading is used in many real-world applications. For example, in web browsers, multiple tabs can run at the same time, and each tab can be handled by a separate thread. This allows users to browse different websites without slowing down the system. 
Another example is mobile applications, where tasks like downloading files, updating the UI, and playing media happen at the same time. Threads help in making the application faster and more responsive.
